# Info
"게임 개발 혼자서 어디까지 할 수 있을까?" 라는 의문에 시작하여 프로그래밍, 디자인, 기획 전부 가능한 만큼 해보는 프로젝트 입니다.
<br>
친숙한 '리그 오브 레전드' 의 조작법과 시스템을 기반으로 전함끼리의 함대전을 구현하는 것이 목표입니다.
<br>
<br>
+ 최대 10 vs 10 환경
+ 최대 1000개 가량의 포탑 + 함재기
+ 위의 조건을 가정하여 발생하는 문제와 이에 대한 솔루션을 고민하기
